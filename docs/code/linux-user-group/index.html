进程凭证

当运行 set-user-ID 程序（即通过调用 exec()将 set-user-ID 程序载入进程的内存中）时，内核会将进程的有效用户 ID 设置为可执行文件的用户 ID。

在 Linux 系统中，set-user-ID 和 set-group-ID 权限位对 shell脚本无效。
