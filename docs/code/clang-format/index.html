  1. 代码风格

基于Google的代码风格（最广泛使用的C++代码风格之一），有以下一些区别

  • 列宽120，Google默认80
  • 4空格缩进，Google默认2，访问控制符不缩进（public, protected, private）
  • 同时设置ConstructorInitializerIndentWidth, ContinuationIndentWidth为8
    • 初始化列表多一次缩进以区别构造函数内的缩进
    • 续行多一次缩进以区别下一行的缩进
  • 指针和引用的位置，指针在右边，引用在左边（纯个人偏好）
  • case不缩进
  • 开括号后不对齐
